# 🍽️ Bittu Cafe – Online Food Ordering System

## 📌 Project Overview

Bittu Cafe is a full-stack food ordering web application developed using ASP.NET Web Forms and SQL Server. The system allows customers to browse menu items, place orders, make payments, and track order status in real time.

---

## 🚀 Features

### Customer Features
- User Registration & Login
- OTP Verification
- Browse Food Menu
- Search Food Items
- Add to Cart
- Online Payment
- Order Tracking
- Order History
- Profile Management

### Order Features
- Dine-In Orders
- Takeaway Orders
- Delivery Orders
- Live Order Status Tracking

### Admin Features
- Manage Categories
- Manage Menu Items
- Manage Orders
- Update Order Status
- Customer Management

---

## ⚙ Setup Instructions

1. Clone the repository
2. Open solution in Visual Studio
3. Create database in SQL Server
4. Execute `BittuCafeDB.sql`
5. Update connection string in `Web.config`
6. Run the application
